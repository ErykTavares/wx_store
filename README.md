# **WX Store**

### **Descrição:**
Recebi um desafio empolgante para uma posição em potencial que envolve a criação de um mini ecommerce com integração de API, bem como a elaboração de uma lista e página de produtos, além de um carrinho de compras funcional. Estou entusiasmado com esta oportunidade única de demonstrar minha criatividade e habilidades técnicas e criar uma solução de qualidade que atenda às expectativas do negócio.
